# Fiction SNS Roadmap

## Phase 0 (Done)

- Django project initialized
- `home` app created
- Root URL connected to `home.index`
- Initial Git/GitHub setup completed

## Phase 1 (Current Target: Web App Foundation)

- Replace plain text top page with template-based layout
- Add basic navigation structure
- Prepare initial domain models (`World`, `Character`, `Post`)
- Implement list/detail pages for minimum reading flow

## Phase 2 (Core Product Flow)

- Create/manage world and character data
- Create/display timeline posts
- Build first usable reading experience

## Phase 3 (AI and Reader Features)

- Character post generation (AI)
- Read progress tracking
- Short summary generation

## Phase 4 (Mobile Expansion)

- Improve responsive UI quality
- Evaluate and add PWA support
- Decide whether native app is required

## Notes

Keep each phase releasable. Avoid implementing too many future features before Phase 1/2 user flow is stable.
