# Milestone Plan

## Milestone 1 (1 to 2 weeks)
Goal:
- Local Django app runs reliably
- Custom top page is shown
- Mobile baseline readability is secured
- Project history is visible through docs and issues

Primary issue set:
- M1-01 to M1-06 in docs/ISSUE_SEEDS.md

Exit criteria:
- Runserver works locally from documented steps
- Root page is custom page (not default Django screen)
- Mobile browser basic check completed
- GitHub has clear issue/commit trace

## Milestone 2 (2 to 4 weeks)
Goal:
- Authentication and basic CRUD for World and Character

Planned outcome:
- Login/logout/register working
- World list/create/edit
- Character create/edit linked to world

## Milestone 3 (3 to 6 weeks)
Goal:
- Human-authored timeline flow for each world

Planned outcome:
- Post model and create form
- Timeline page sorted by newest
- Mobile-first timeline readability

## Milestone 4 (2 to 3 weeks)
Goal:
- Public deployment for smartphone access

Planned outcome:
- Hosted URL and end-to-end check from phone

## Milestone 5+
Goal:
- AI-assisted generation and reading support

Planned outcome:
- Character auto post generation
- Read progress and summary support
